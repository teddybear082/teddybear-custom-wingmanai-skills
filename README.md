# teddybear-custom-wingmanai-skills
 Repo to house WIP or non-official wingmanai skills

Download the latest release zips here: https://github.com/teddybear082/teddybear-custom-wingmanai-skills/releases/tag/wingman_2_1_custom_skills

Then unzip it to your wingman user data custom skills folder found at (for windows users):

```C:\Users\{YOUR USERNAME}\AppData\Roaming\ShipBit\WingmanAI\custom_skills```

Note, one skill included in here I DID NOT CREATE, all credit goes to user **sneepe** on discord for the **Natural Comments** skill.
